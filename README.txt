<------------------------------------------   Explicação do Software    ------------------------------------------>

Antes de tudo quero informar que software utiliza um banco de dados local na empresa onde trabalho atualmente.

A finalidade deste software é facilitar o cotidiano dos colaboradores da Duarte Contabilidade, tanto o departamento de contabilidade como o departamento fiscal.
Quando recebemos o financeiro dos nossos clientes, esses arquivos(Txt's, excel e PDF) passam por este software para que seja feito um tratamento com ele
e depois gerando um layout para importação do mesmo no software da Wolters Kluwer, o Prosoft.
Agora estamos criando um módulo que irá dar um avanço ao software, que é a Conciliação Contábil, que ainda está em desenvolvimento.
